Comparison Operators

== equal to (equality) [it does implicit (automatic) conversion]

=== equal value and equal type (strict equality) [you have to do explicit (manual convert string to number) conversion] recommended process

!= not equal [it does implicit (automatic) conversion]

!== not equal value or not equal type (strict equality) [you have to do explicit (manual convert string to number) conversion] recommended process

> greater than [it does implicit (automatic) conversion]

< less than [it does implicit (automatic) conversion]

>= greater than or equal to [it does implicit (automatic) conversion]

<= less than or equal to [it does implicit (automatic) conversion]

Please check comment in the script file for details explanation.
