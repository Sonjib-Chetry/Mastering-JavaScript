Welcome to my **90 Days of JavaScript Revision Challenge**!

Please read the script file and understand the project and try to build your own.

In this mini project, you'll build a favorite icon toggler by utilizing JavaScript click events.

User Stories

  • You should have an unordered list with three items.
  • The unordered list should have the class item-list.
  • The three list items should contain the item name followed by a button element with the class favorite-icon.
  • The button element should contain the code &#9825; initially to represent an empty heart.
  • When a button element containing a heart is clicked, you should add a class named filled to the clicked button if it's not already present, and remove it if it is.
  • You should have a .filled class selector that sets some CSS properties (e.g., color).
  • When a heart button is clicked, it should toggle between &#9825; (empty heart) and &#10084; (filled heart), depending on its current state.

Test Cases

  1. You should have an unordered list.
  2. Your unordered list should have 3 items.
  3. Your unordered list should have the class item-list.
  4. Your individual list items should contain the item name.
  5. Your individual list items should contain a button element with the class favorite-icon.
  6. Initially, the button elements should contain the code &#9825; to represent an empty heart.
  7. You should have a .filled selector that sets some CSS properties.
  8. When a button element is clicked, and it contains the class filled, you should remove the class and change the innerHTML of the button to &#9825;.
  9. When a button element is clicked, and it doesn't contain the class filled, you should add the class and change the innerHTML of the button to &#10084;.
