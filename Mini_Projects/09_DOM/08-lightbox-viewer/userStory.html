Welcome to my **90 Days of JavaScript Revision Challenge**!

Please read the script file and understand the project and try to build your own.

In this mini project, you'll build a lightbox viewer for viewing images in a focused mode. You'll practice click events and toggling classes.

User Stories

  • You should have a <div> with a class of gallery inside your <body> element.
  • Inside the .gallery div, you should have three image thumbnails, each with the class gallery-item.
  • The three thumbnail images should use the following URLs:
    • https://cdn.freecodecamp.org/curriculum/labs/stonehenge-thumbnail.jpg
    • https://cdn.freecodecamp.org/curriculum/labs/storm-thumbnail.jpg
    • https://cdn.freecodecamp.org/curriculum/labs/trees-thumbnail.jpg
  • You should have a <div> with the class lightbox inside your body.
  • Inside the .lightbox div, include:
    • A <button> element with the id close-btn (suggested text: ×)
    • An <img> element with the id lightbox-image
  • The .lightbox element should:
    • Have position: fixed
    • Cover the full viewport using height: 100% and width: 100%
    • Be aligned at the top-left corner (e.g., top: 0; left: 0;)
    • Have a background color
    • Be hidden initially using display: none
  • When a .gallery-item image is clicked:
    • The .lightbox should become visible by setting its display to flex
    • The #lightbox-image's src should be updated by removing -thumbnail from the clicked image's URL
  • Clicking either the #close-btn or the .lightbox background should hide the lightbox by setting its display back to none.

Test Cases

  1. There should be a div with the class gallery.
  2. The .gallery should contain three img elements with the class gallery-item.
  3. The three image elements should have the correct src values for thumbnails.
  4. There should be a div with the class lightbox.
  5. The .lightbox should contain a button with id close-btn.
  6. The .lightbox should contain an img with id lightbox-image.
  7. The .lightbox should have position: fixed.
  8. The .lightbox should be set to height: 100% and width: 100%.
  9. The .lightbox should be aligned at the top left with top: 0 and left: 0.
  10. The .lightbox should have a visible background color.
  11. The .lightbox should be hidden initially (i.e., display: none).
  12. Clicking a thumbnail should change .lightbox to display: flex.
  13. Clicking a thumbnail should set #lightbox-image's src to the full-size image version (without -thumbnail).
  14. Clicking the #close-btn should hide the lightbox.
  15. Clicking the background of the .lightbox should also hide the lightbox.
