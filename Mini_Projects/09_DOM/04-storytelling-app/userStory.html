Welcome to my **90 Days of JavaScript Revision Challenge**!

Please read the script file and understand the project and try to build your own.

In this mini project, you will build a storytelling app that will allow you to list different stories based on genre.

User Stories

  • You should create a container element with class story-container to display the story and hold the buttons.
  • You should define a paragraph element with ID result where the selected story will be shown.
  • You should add three buttons with the IDs scary-btn, funny-btn, and adventure-btn for selecting each story type.
  • You should declare a constant storyObj containing an object with three properties: scary, funny, and adventure.
  • Each property in storyObj should itself be an object containing a story string and a borderColor string.
  • You should define a function displayStory that accepts a genre string and updates the #result paragraph with the corresponding story text.
  • The displayStory function should also update the border color of the .story-container using the borderColor value from storyObj.
  • You should add event listeners to each button that call displayStory with their respective genre strings.
  • If an unknown genre is passed to displayStory, it should display Story not found. and remove any border color styling.

Test Cases

  1. storyObj should be an object containing the keys scary, funny, and adventure.
  2. Each genre in storyObj should have a story and borderColor property.
  3. When the "Scary Story" button is clicked, #result should display the scary story text, and the container's border color should be #ee4b2b.
  4. When the "Funny Story" button is clicked, #result should display the funny story, and the border color should be #f1be32.
  5. When the "Adventure Story" button is clicked, #result should display the adventure story, and the border color should be #acd157.
  6. If displayStory is called with an undefined genre (e.g., "romantic"), #result should show Story not found. and the border color should reset (transparent or none).
  7. All three buttons should correctly trigger the displayStory function with the appropriate genre string.
  8. The paragraph #result should reflect the last selected story and update every time a different button is clicked.
  9. The .story-container should visually update its border color based on the selected story genre.
