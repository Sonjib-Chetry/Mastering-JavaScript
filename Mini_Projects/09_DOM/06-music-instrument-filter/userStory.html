Welcome to my **90 Days of JavaScript Revision Challenge**!

Please read the script file and understand the project and try to build your own.

In this mini project, you will build a music instrument filter with JavaScript.

User Stories

  • You should have an HTML page titled Music Instrument Filter.
  • You should display a heading that says Student Instruments.
  • You should have a <select> dropdown menu for filtering instrument categories.
  • The dropdown should include the following options: All, Woodwinds, Brass, and Percussion.
  • You should display a list of instrument cards inside a container with class products-container.
  • Each card should show the name of the instrument and its price.
  • You should have a JavaScript array of instrument objects, each with properties: category, instrument, and price.
  • You should have a function instrumentCards that filters and returns the matching instrument cards as a string of HTML.
  • When the user selects an option from the dropdown, the displayed instrument cards should update based on the selected category.

Test Cases

  1. The HTML should include a <select> element for filtering instruments.
  2. The dropdown should have four <option> elements with values all, woodwinds, brass, and percussion.
  3. The initial list should show all 10 instruments.
  4. When "Brass" is selected, only 3 brass instruments should be displayed.
  5. When "Woodwinds" is selected, only 3 woodwind instruments should be displayed.
  6. When "Percussion" is selected, only 4 percussion instruments should be displayed.
  7. The instrument cards should be updated dynamically without refreshing the page.
  8. The function instrumentCards("brass") should return HTML for only brass instruments.
  9. The function instrumentCards("all") should return HTML for all instruments.
  10. Each card should include an <h2> for the instrument name and a <p> for the price in the format $PRICE.
