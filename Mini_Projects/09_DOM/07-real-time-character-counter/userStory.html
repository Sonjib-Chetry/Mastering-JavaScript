Welcome to my **90 Days of JavaScript Revision Challenge**!

Please read the script file and understand the project and try to build your own.

In this mini project, you'll build a real-time character counter You'll practice how to work with the input event when the user types in the input field.

User Stories

  • You should have a <textarea> element with the id of text-input.
  • You should have a <p> element with the id of char-count.
  • The #char-count element should initially contain the text "Character Count: 0/50".
  • As the user types in the textarea, the #char-count should update to reflect the current number of characters entered.
  • When the user types "hello coder", the character count should display "Character Count: 11/50".
  • When the user types "i am a programmer", the character count should display "Character Count: 17/50".
  • When the user types "hello world", the character count should display "Character Count: 11/50".
  • When the user types "I am learning a new language and it is called c++.", the character count should display "Character Count: 50/50".
  • The user should not be able to enter more than 50 characters into the textarea.
  • When the character count reaches 50, the text inside #char-count should be displayed in red.
  • All logic should be handled with JavaScript only. You should not use the maxlength attribute.
