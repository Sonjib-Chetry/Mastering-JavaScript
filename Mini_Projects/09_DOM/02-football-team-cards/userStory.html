Welcome to my **90 Days of JavaScript Revision Challenge**!

Please read the script file and understand the project and try to build your own.

In this mini project, you will build a set of football team cards. The user should be able to use the dropdown menu and filter between the different players based on their positions.

User Stories

  • You should create a JavaScript object named footballTeam with the following properties:
    • team: a string representing the name of the team.
    • year: a number representing the year.
    • headCoach: a string containing the head coach's name.
    • players: an array with at least four player objects.
  • Each object in the players array should include:
    • name: a string with the player’s name.
    • position: a string with one of the following values: "forward", "midfielder", "defender", or "goalkeeper".
    • isCaptain: a boolean indicating whether the player is the captain. Only one player should have true.
  • You should display the values of headCoach, team, and year in the HTML elements with ids: head-coach, team, and year respectively.
  • You should display each player as a div with class player-card inside the #player-cards element. Each player card should contain:
    • An <h2> with the player’s name. If the player is captain, prepend it with "(Captain)".
    • A <p> with the player’s position, e.g., Position: midfielder.
  • You should have a dropdown filter to show players by their position (forward, midfielder, defender, goalkeeper, or all).
  • When the dropdown selection changes, only the players matching the selected position should be displayed. If "All Players" is selected, display all players.

Test Cases

  1. You should declare a variable footballTeam in your script.
  2. footballTeam should be an object with four properties: team, year, headCoach, and players.
  3. The team property should be a string.
  4. The year property should be a number.
  5. The headCoach property should be a string.
  6. The players property should be an array of at least four objects.
  7. Each player object should have the properties name (string), position (valid string), and isCaptain (boolean).
  8. Only one player should have isCaptain set to true.
  9. The values of team, year, and headCoach should appear in the correct HTML elements by their IDs.
  10. All players should be displayed when "All Players" is selected from the dropdown.
  11. Only players with the selected position should be displayed when filtering by "Forward".
  12. Only players with the selected position should be displayed when filtering by "Midfielder".
  13. Only players with the selected position should be displayed when filtering by "Defender".
  14. Only players with the selected position should be displayed when filtering by "Goalkeeper".
