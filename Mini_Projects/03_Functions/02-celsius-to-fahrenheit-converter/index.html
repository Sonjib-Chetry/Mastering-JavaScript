In this mini project you will implement a function that converts the temperature from Celsius to Fahrenheit.

User Story

You should create a function named convertCtoF.
- The function should take a single numeric argument, which is the temperature in Celsius.
- The function should return the temperature converted to Fahrenheit as a number.

Test Cases

  • convertCtoF(0) should return a number.
  • convertCtoF(-30) should return -22.
  • convertCtoF(-10) should return 14.
  • convertCtoF(0) should return 32.
  • convertCtoF(20) should return 68.
  • convertCtoF(30) should return 86.
