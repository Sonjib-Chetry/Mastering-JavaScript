Welcome to my **90 Days of JavaScript Revision Challenge**!

Please read the script file and understand the project and try to build your own.

In this mini project, you will learn JavaScript fundamentals by building a greeting bot. You will learn about variables, let, const, console.log and basic string usage.

User Stories:

  • You should declare a variable named bot and assign it a string value.
  • You should declare a variable named botLocation and assign it a string value representing a location.
  • You should log the greeting message "I am excited to talk to you." to the console.
  • You should log the message "Allow me to introduce myself." to the console.
  • You should declare a variable botIntroduction and assign it a string that introduces the bot using the bot variable.
  • You should log the botIntroduction to the console.
  • You should declare a variable botLocationSentence and assign it a string that includes the botLocation value.
  • You should log the botLocationSentence to the console.
  • You should reassign the bot variable to a new value "professorBot" and declare a variable nicknameIntroduction that uses this new bot value.
  • You should log nicknameIntroduction to the console.
  • You should reassign the bot variable again to "awesomeTeacherBot" and declare a variable newNicknameGreeting that uses this new value.
  • You should log newNicknameGreeting to the console.
  • You should declare a variable favoriteSubject and assign it a string value.
  • You should declare a variable favoriteSubjectSentence and assign it a sentence including the favoriteSubject.
  • You should log favoriteSubjectSentence to the console.
  • You should log the goodbye message "Well, it was nice to talk to you. Have a nice day!" to the console.

Tests

  1. The first console log should output "I am excited to talk to you.".
  2. The second console log should output "Allow me to introduce myself.".
  3. The third console log should output "My name is teacherBot.".
  4. The fourth console log should output "I live in the universe.".
  5. The fifth console log should output "My nickname is professorBot.".
  6. The sixth console log should output "I love my nickname but I wish people would call me awesomeTeacherBot.".
  7. The seventh console log should output "My favorite subject is Computer Science.".
  8. The final console log should output "Well, it was nice to talk to you. Have a nice day!".
