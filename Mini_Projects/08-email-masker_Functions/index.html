Welcome to my **90 Days of JavaScript Revision Challenge**!

Please read the script file and understand the project and try to build your own.

In this mini project, you'll build an email masker that will take an email address and obscure it. You'll practice string slicing, concatenation, and using functions.

Challenge:-

Create a function named maskEmail that takes email as an argument.

Inside the function, you should mask the email and append the domain name to it. Remember that you can use methods like slice, repeat, indexOf, or even replace to help you.

Outside the function, declare a variable named email to store the email address you want to mask.

Call the maskEmail function with the email variable and output the result to the console.

Examples

      
maskEmail("apple.pie@example.com")     // returns "a*******e@example.com"
maskEmail("freecodecamp@example.com") // returns "f**********p@example.com"
maskEmail("info@test.dev")            // returns "i**o@test.dev"
maskEmail("user@domain.org")          // returns "u**r@domain.org"
