Welcome to my **90 Days of JavaScript Revision Challenge**!

Please read the script file and understand the project and try to build your own.

In this mini project, you will continue to learn more about JavaScript strings by building a chatbot. You will learn how to work with template literals, and the indexOf method.

User Stories:

  1. You should log "Hi there!" to the console as a greeting message.
  2. You should declare a variable botName and assign it a value (e.g., "teacherBot").
  3. You should use a template literal to assign greeting the value "My name is ${botName}.".
  4. You should log greeting to the console.
  5. You should declare and assign values to subject and topic.
  6. You should declare sentence and assign it a template literal about learning the topic in the subject.
  7. You should log sentence to the console.
  8. You should create strLengthIntro using a template literal that mentions using the length property on subject, and log it.
  9. You should log the length of subject.
  10. You should log a message about using length on topic, and log the length of topic.
  11. You should log a message about accessing the first letter of subject, then log the first character.
  12. You should log a message about accessing the second letter of subject, then log the second character.
  13. You should log a message about accessing the last letter of subject.
  14. You should assign lastCharacter to the last letter of subject using subject[subject.length - 1] and log it.
  15. You should declare and assign the string "Learning is fun." to learningIsFunSentence.
  16. You should log a message about finding substrings in learningIsFunSentence.
  17. You should log the results of indexOf for the substrings "Learning", "fun", and "learning" from learningIsFunSentence.
  18. You should end with the message "I hope you enjoyed learning today.".

Tests:

  1. You should log "Hi there!" to the console.
  2. You should declare botName and assign it a string.
  3. You should create greeting using template literals and log it.
  4. You should declare and assign string values to subject and topic.
  5. sentence should be a template literal that includes topic and subject and be logged.
  6. strLengthIntro should use a template literal that references subject and be logged.
  7. You should correctly log the length of subject using .length.
  8. You should log a similar message and the length of topic.
  9. You should log the first letter of subject using bracket notation and the message before it.
  10. You should log the second letter of subject using bracket notation and the message before it.
  11. You should log a message about the last letter, assign lastCharacter using subject[subject.length - 1], and log it.
  12. You should declare and assign learningIsFunSentence the value "Learning is fun.".
  13. You should log a message about finding substring positions.
  14. You should use indexOf to log the positions of "Learning", "fun", and "learning" in learningIsFunSentence.
  15. You should log "I hope you enjoyed learning today." to the console.
