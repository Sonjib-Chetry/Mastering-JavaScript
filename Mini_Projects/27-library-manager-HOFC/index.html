In this mini project, you will learn higher order array methods by building a library manager.

  1. Library Structure:
    • A constant library should be declared as an array of objects.
    • Each object should include the following properties: title, author, about, and pages.
  2. Book Information:
    • Create a function named getBookInformation.
    • This function should take an array as a parameter.
    • It should return a string that lists book titles and authors in the format: "Title by Author", each on a new line.
  3. Book Summaries:
    • Create a function named getBookSummaries.
    • It should return a string containing the about field of each book, separated by line breaks.
  4. Filter by Author:
    • Create a function named getBooksByAuthor.
    • This function should accept the library array and an author name.
    • It should return a list of book titles by that author, each on a new line.
    • If no books match the author, return an empty string.
  5. Total Pages:
    • Create a function named getTotalPages.
    • It should return the total number of pages by summing the pages field of all books in the catalog.

Tests

  • 1. You should declare a constant library that is an array.
  • 2. Each object in library should have title, author, about, and pages keys.
  • 3. You should have a function named getBookInformation.
  • 4. getBookInformation(library) should return a string.
  • 5. The returned string should include all titles and authors from the catalog formatted as [title] by [author].
  • 6. You should have a function named getBookSummaries.
  • 7. getBookSummaries(library) should return a string containing all about fields separated by new lines.
  • 8. You should have a function named getBooksByAuthor.
  • 9. getBooksByAuthor(library, "Arvid Kahl") should return a string containing two titles.
  • 10. getBooksByAuthor(library, "James Clear") should return the string Atomic Habits.
  • 11. getBooksByAuthor(library, "Unknown Author") should return an empty string.
  • 12. You should have a function named getTotalPages.
