Welcome to my **90 Days of JavaScript Revision Challenge**!

Please read the script file and understand the project and try to build at your own.

As you expand your JavaScript skills, you'll want to get comfortable with array manipulation methods, such as map(), reduce(), and filter(). In this statistics calculator project, you'll gain experience with handling user input, DOM manipulation, and method chaining. You'll get practice by performing statistical calculations like mean, median, mode, variance, and standard deviation.

üìù User Stories

  • You should have an input element with the id numbers. This input will accept a comma-separated list of numbers (e.g., 1, 2, 3, 4).
  • You should have a button element that triggers the calculation when clicked.
  • You should have output elements (e.g., <span> or <p>) with the following ids to display results:
    • mean
    • median
    • mode
    • range
    • variance
    • standardDeviation
  • In JavaScript:
    • Implement a function getMean(array) that calculates the average of numbers.
    • Implement a function getMedian(array) that finds the middle value (or average of two middle values).
    • Implement a function getMode(array) that finds the most frequent value(s). If no mode exists, return "No mode value".
    • Implement a function getRange(array) that returns the difference between the maximum and minimum values.
    • Implement a function getVariance(array) that calculates the average of squared differences from the mean.
    • Implement a function getStandardDeviation(array) that is the square root of variance.
  • Create a calculate() function that:
    • Reads the input from #numbers.
    • Parses it into an array of valid numbers.
    • Calculates mean, median, mode, range, variance, and standard deviation.
    • Updates the respective output elements with the calculated values.
