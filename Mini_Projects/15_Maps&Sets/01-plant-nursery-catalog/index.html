  1. Create several plant objects. Each plant object should have:
    • commonName (string)
    • scientificName (string)
    • cultivar (string)
  2. Create a Map called catalog where:
    • The key is a plant object
    • The value is an object with keys small, medium, large representing stock counts
  3. Implement a function removePlant(plantObject) that:
    • Removes the plant from the catalog
    • Returns true if successfully removed, false otherwise
  4. Implement a function sellPlants(plantObject, potSize, numberOfPots) that:
    • Checks if the plant exists in catalog
    • If not, returns the string "Item not found."
    • If stock for the given size is less than requested, returns a string like:
      Not enough <potSize> size pots for <scientificName> '<cultivar>'. Only <available> left.
    • If enough stock is available, subtracts the sold quantity from the correct size and updates catalog
    • Returns "Catalog successfully updated." on success

  6. Implement a function displayPlantsSet() that:
    • Collects all common names from the catalog
    • Returns them as a Set (no duplicates)
