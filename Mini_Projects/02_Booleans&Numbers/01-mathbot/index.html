Welcome to my **90 Days of JavaScript Revision Challenge**!

Please read the script file and understand the project and try to build your own.

In this mini project, you will continue to learn more about JavaScript strings by building a chatbot. You will learn how to work with template literals, and the indexOf method.

User Stories:

  1. You should declare a variable botName and assign it a value (e.g., "MathBot").
  2. You should declare a variable greeting using template literals to create a greeting message including botName, and log it to the console.
  3. You should log a message describing the purpose of Math.random().
  4. You should declare randomNum, assign it a value using Math.random(), and log it.
  5. You should log a message indicating that a random number will be generated between two values.
  6. You should declare min and max and assign numeric values (e.g., 1 and 100).
  7. You should declare randomNum2 and assign it a random number between min and max using the formula Math.random() * (max - min) + min, and log it.
  8. You should log a message about Math.floor(), then declare numRoundedDown, assign it a rounded-down value using Math.floor(), and log it.
  9. You should log a message about Math.ceil(), then declare numRoundedUp, assign it a rounded-up value using Math.ceil(), and log it.
  10. You should log a message about Math.round(), then declare and log two variables, numRounded and numRounded2, using Math.round().
  11. You should log a message explaining Math.max() and Math.min() methods.
  12. You should declare maxNum using Math.max() on a list of numbers and log it.
  13. You should declare minNum using Math.min() on a list of numbers and log it.
  14. You should log a farewell message indicating that the bot enjoyed teaching about Math methods.

Tests:

  1. You should declare botName and assign it the string "MathBot".
  2. You should declare greeting using a template literal that includes botName and log it.
  3. You should log a message explaining Math.random().
  4. You should declare randomNum using Math.random() and log it.
  5. You should declare min and max with numeric values and use them to calculate randomNum2.
  6. randomNum2 should be calculated with Math.random() * (max - min) + min and logged.
  7. You should explain and demonstrate Math.floor() with numRoundedDown.
  8. You should explain and demonstrate Math.ceil() with numRoundedUp.
  9. You should explain and demonstrate Math.round() with both numRounded and numRounded2.
  10. You should explain and demonstrate Math.max() with maxNum.
  11. You should explain and demonstrate Math.min() with minNum.
  12. You should log the farewell message: "It was fun learning about the different Math methods with you!".
