Welcome to my **90 Days of JavaScript Revision Challenge**!

Please read the script file and understand the project and try to build at your own.

You are creating a function that aids in the maintenance of a musical album collection. The collection is organized as an object that contains multiple albums which are also objects. Each album is represented in the collection with a unique id as the property name. Within each album object, there are various properties describing information about the album. Not all albums have complete information.

User Story

You need to write a function called updateRecords(records, id, prop, value) that manipulates a record collection object.

Function Parameters:

  • records - an object containing several individual albums.
  • id - a number representing a specific album in the records object.
  • prop - a string representing the name of the album’s property to update.
  • value - a string containing the information used to update the album’s property.

Rules:

  • Your function must always return the entire records object.
  • If value is an empty string, delete the given prop property from the album.
  • If prop isn't tracks and value isn't an empty string, assign value to that album's prop.
  • If prop is tracks and value isn't an empty string, but the album doesn’t have a tracks property, create an empty array and add value to it.
  • If prop is tracks and value isn't an empty string, add value to the end of the album's existing tracks array.

Test Cases

  • You should have an updateRecords function.
  • updateRecords(recordCollection, 5439, "artist", "ABBA") → artist should be the string "ABBA".
  • updateRecords(recordCollection, 5439, "tracks", "Take a Chance on Me") → tracks should have "Take a Chance on Me" as the last and only element.
  • updateRecords(recordCollection, 2548, "artist", "") → artist should be removed.
  • updateRecords(recordCollection, 1245, "tracks", "Addicted to Love") → tracks should include "Addicted to Love".
  • updateRecords(recordCollection, 2468, "tracks", "Free") → tracks should still contain "1999" as the first element.
  • updateRecords(recordCollection, 2548, "tracks", "") → tracks should be removed.
  • updateRecords(recordCollection, 1245, "albumTitle", "Riptide") → albumTitle should be "Riptide".
