Welcome to my **90 Days of JavaScript Revision Challenge**!

Please read the script file and understand the project and try to build your own.

In this mini project, you'll build a quiz game using JavaScript arrays and objects. You'll also practice using functions to randomly select a question and an answer from an array and compare them.

User Stories – Quiz Program

  • You should create an array named questions.
  • The questions array should contain at least five objects, each having the following keys:
    • category: a string representing a question category.
    • question: a string representing the question itself.
    • choices: an array of three strings, which are the answer options.
    • answer: a string that is the correct answer (must be one of the items in choices).
  • You should have a function named getRandomQuestion that:
    • Takes an array of questions as a parameter.
    • Returns a random question object from the array.
  • You should have a function named getRandomComputerChoice that:
    • Takes the array of available choices as a parameter.
    • Returns a random answer from the choices.
  • You should have a function named getResults that:
    • Takes the question object as the first parameter.
    • Takes the computer's choice as the second parameter.
    • Returns The computer's choice is correct! if the answer is correct.
    • If incorrect, returns The computer's choice is wrong. The correct answer is: <correct-answer>.

Test Cases – Quiz Program

  • You should create an array named questions.
  • The questions array should contain at least five objects, each having the keys category, question, choices, and answer.
  • The category key should have the value of a string representing a question category.
  • The question key should have the value of a string representing a question.
  • The choices key should have the value of an array containing three strings.
  • The answer key should have the value of a string.
  • The value of answer should be included in the choices array.
  • You should have a function named getRandomQuestion that:
    • Takes an array of questions as a parameter.
    • Returns a random question object from the array.
  • You should have a function named getRandomComputerChoice that:
    • Takes the array of available choices as a parameter.
    • Returns a random answer to the selected question.
  • You should have a function named getResults.
  • Your getResults function should take:
    • The question object as the first parameter.
    • The computer's choice as the second parameter.
  • If the computer choice matches the answer, getResults should return The computer's choice is correct!.
  • If the computer choice doesn't match the answer, getResults should return The computer's choice is wrong. The correct answer is: <correct-answer>, where <correct-answer> is the value of the correct answer to the chosen question.
