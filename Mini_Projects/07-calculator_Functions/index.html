Welcome to my **90 Days of JavaScript Revision Challenge**!

Please read the script file and understand the project and try to build your own.

In this mini project, you will review your knowledge of functions by building a calculator.

User Stories

  • You should have a function named addTwoAndSeven that returns the sum of 2 and 7.
  • You should log the result of calling addTwoAndSeven() to the console.
  • You should have a function named addThreeAndFour that returns the sum of 3 and 4.
  • You should log the result of calling addThreeAndFour() to the console.
  • You should have a function named calculateSum that takes two parameters and returns their sum.
  • You should call calculateSum multiple times with different arguments and log the results to the console.
  • You should have a function named calculateDifference that takes two parameters and returns their difference (num1 - num2).
  • You should call calculateDifference multiple times and log the results to the console.
  • You should have a function named calculateProduct that takes two numbers and returns their product.
  • You should call calculateProduct and log the result to the console.
  • You should have a function named calculateQuotient that takes two numbers and returns the quotient of the first divided by the second.
  • If the second number is 0, calculateQuotient should return the string "Error: Division by zero".
  • You should call calculateQuotient with valid and zero divisors and log the results to the console.
  • You should have a function named calculateSquare that returns the square of a number using Math.pow().
  • You should call calculateSquare with different arguments and log the results to the console.
  • You should have a function named calculateSquareRoot that returns the square root of a number using Math.sqrt().
  • You should call calculateSquareRoot with different arguments and log the results to the console.

Test Cases

  1. Calling addTwoAndSeven() should return 9.
  2. Calling addThreeAndFour() should return 7.
  3. Calling calculateSum(2, 5) should return 7.
  4. Calling calculateSum(10, 10) should return 20.
  5. Calling calculateSum(5, 5) should return 10.
  6. Calling calculateDifference(22, 5) should return 17.
  7. Calling calculateDifference(12, 1) should return 11.
  8. Calling calculateDifference(17, 9) should return 8.
  9. Calling calculateProduct(13, 5) should return 65.
  10. Calling calculateQuotient(7, 11) should return approximately 0.636.
  11. Calling calculateQuotient(3, 0) should return "Error: Division by zero".
  12. Calling calculateSquare(2) should return 4.
  13. Calling calculateSquare(9) should return 81.
  14. Calling calculateSquareRoot(25) should return 5.
  15. Calling calculateSquareRoot(100) should return 10.
