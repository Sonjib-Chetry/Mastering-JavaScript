Welcome to my **90 Days of JavaScript Revision Challenge**!

Please read the script file and understand the project and try to build at your own.

In this mini project, you will use JavaScript fundamentals to create a function that finds the largest number in each sub-array of a given array.

User Stories

  • You should create a function named largestOfAll.
  • The function should take a single parameter, which is an array of arrays.
  • The function should return a new array containing the largest number from each sub-array.

Test Cases

  • You should have a function named largestOfAll.
  • largestOfAll should take a single parameter.
  • largestOfAll([[4, 5, 1, 3], [13, 27, 18, 26], [32, 35, 37, 39], [1000, 1001, 857, 1]]) → [5, 27, 39, 1001].
  • largestOfAll([[13, 27, 18, 26], [4, 5, 1, 3], [32, 35, 37, 39], [1000, 1001, 857, 1]]) → [27, 5, 39, 1001].
  • largestOfAll([[4, 9, 1, 3], [13, 35, 18, 26], [32, 35, 97, 39], [1000000, 1001, 857, 1]]) → [9, 35, 97, 1000000].
  • largestOfAll([[17, 23, 25, 12], [25, 7, 34, 48], [4, -10, 18, 21], [-72, -3, -17, -10]]) → [25, 48, 21, -3].
