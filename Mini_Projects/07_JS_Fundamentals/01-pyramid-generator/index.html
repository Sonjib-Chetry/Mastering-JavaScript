Welcome to my **90 Days of JavaScript Revision Challenge**!

Please read the script file and understand the project and try to build your own.

In this mini project you'll build a pyramid generator. You'll take a number as input and generate a pyramid with that many levels using a loop.

User Stories – pyramid Function

  • You should define a function named pyramid that takes three arguments.
  • The arguments should be:
    • A string representing the character to be used in the pyramid.
    • An integer representing the number of rows.
    • A Boolean indicating the orientation (upward or downward) of the pyramid.
  • If the third argument is false, the pyramid should have its vertex facing upward.
  • If the third argument is true, the pyramid should have its vertex facing downward.
  • The pyramid should follow these rules:
    • The first/last row (depending on direction) has 1 character.
    • Each subsequent/previous row has 2 more characters than the previous one.
    • Each row should be center-aligned by adding leading spaces.
    • The final string should start and end with a newline character.
  • Example: pyramid("o", 4, false) should return: 
       o
  ooo
 ooooo
ooooooo

Test Cases – pyramid Function

  • pyramid("o", 4, false) should return "\n o\n ooo\n ooooo\nooooooo\n"
  • pyramid("p", 5, true) should return "\nppppppppp\n ppppppp\n ppppp\n ppp\n p\n"

  • The pyramid function should have three parameters:
