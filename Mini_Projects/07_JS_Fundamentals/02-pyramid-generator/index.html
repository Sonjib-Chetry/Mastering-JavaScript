Welcome to my **90 Days of JavaScript Revision Challenge**!

Please read the script file and understand the project and try to build at your own.

In this practice project, you'll learn fundamental programming concepts in JavaScript by coding your own Pyramid Generator. You'll learn how to work with arrays, strings, functions, loops, if/else statements, and more.

ðŸŽ¯ User Stories

  • You should define a constant character to represent the symbol used to build the pyramid.
  • You should define a constant count to represent the total number of rows in the pyramid.
  • You should define an array rows to hold each row of the pyramid as strings.
  • You should define a boolean variable inverted to control whether the pyramid is normal or inverted.
  • You should create a function padRow that:
    • Takes two arguments: rowNumber (the current row) and rowCount (total rows).
    • Returns a string that contains spaces and repeated characters so that the pyramid row is correctly aligned.
    • Each row should end with a newline character (\n).
  • You should use a for loop from 1 to count to generate pyramid rows.
  • If inverted is false, push rows to the end of the rows array.
  • If inverted is true, unshift rows to the beginning of the rows array (so the pyramid is upside down).
  • You should combine all rows into a single string result.
  • You should console.log the final pyramid stored in result.
