Welcome to my **90 Days of JavaScript Revision Challenge**!

Please read the script file and understand the project and try to build at your own.

In this mini project, you will create a function that looks through an array and returns the first element in it that passes a "truth test".

User Stories

  • You should have a function named findElement.
  • The findElement function should accept two parameters: an array and a function.
  • The function should return the first element in the array that passes the truth test provided by the function.
  • If no element passes the truth test, findElement should return undefined.

Test Cases

  • You should have a findElement function.
  • findElement([1, 3, 5, 8, 9, 10], function(num) { return num % 2 === 0; }) → 8.
  • findElement([1, 3, 5, 9], function(num) { return num % 2 === 0; }) → undefined.
  • findElement([1, 2, 3, 4], function(num) { return num > 2; }) → 3.
  • findElement(["hello", "world", "javascript"], function(str) { return str.length > 5; }) → "javascript".
  • findElement(["cat", "dog", "bird"], function(str) { return str.length > 10; }) → undefined.
  • findElement([2, 4, 6, 8], function(num) { return num % 2 === 0; }) → 2.
  • findElement([], function(num) { return num > 0; }) → undefined.
