Welcome to my **90 Days of JavaScript Revision Challenge**!

Please read the script file and understand the project and try to build your own.

In this mini project, you create a function that reverses a given string.

User Stories – reverseString Function

  • You should create a function named reverseString that takes a string as an argument.
  • The function should return the reversed string.

reverseString – Test Cases

  • You should have a function named reverseString.
  • reverseString should take a string as an argument.
  • reverseString("hello") should return a string.
  • reverseString("hello") should return the string olleh.
  • reverseString("Howdy") should return the string ydwoH.
  • reverseString("Greetings from Earth") should return the string htraE morf sgniteerG.
