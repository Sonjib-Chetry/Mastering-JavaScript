Welcome to my **90 Days of JavaScript Revision Challenge**!

Please read the script file and understand the project and try to build your own.

In this mini project, you'll review working with arrays and random numbers by building a lunch picker program.

Lunch Menu Manager – User Stories

  • You should create a variable named lunches and assign it an empty array that will be used to store lunch items.
  • You should create a function addLunchToEnd that takes:
    • an array as the first argument,
    • a string as the second argument.
    The function should:
    • Add the string to the end of the array.
    • Log "[Lunch Item] added to the end of the lunch menu." to the console.
    • Return the updated array.
  • You should create a function addLunchToStart with the same argument pattern as above. It should:
    • Add the string to the start of the array.
    • Log "[Lunch Item] added to the start of the lunch menu." to the console.
    • Return the updated array.
  • You should create a function removeLastLunch that takes an array as its only argument. It should:
    • Remove the last element from the array.
    • If successful, log "[Lunch Item] removed from the end of the lunch menu.".
    • If the array is empty, log "No lunches to remove.".
    • Return the updated array.
  • You should create a function removeFirstLunch that takes an array as its only argument. It should:
    • Remove the first element from the array.
    • If successful, log "[Lunch Item] removed from the start of the lunch menu.".
    • If the array is empty, log "No lunches to remove.".
    • Return the updated array.
  • You should create a function getRandomLunch that takes an array as its argument. It should:
    • Select a random element from the array.
    • If successful, log "Randomly selected lunch: [Lunch Item]".
    • If the array is empty, log "No lunches available.".
  • You should create a function showLunchMenu that takes an array as its argument. It should:
    • If the array has items, log "Menu items: [Lunch Item], [Lunch Item], ...".
    • If the array is empty, log "The menu is empty.".

Lunch Menu – Test Cases

  • You should declare a variable lunches and assign it an empty array to store the lunch items.
  • You should define a function addLunchToEnd.
  • The function addLunchToEnd should have two parameters.
  • addLunchToEnd(lunches, "Tacos") should log "Tacos added to the end of the lunch menu." to the console.
  • addLunchToEnd(["Pizza", "Tacos"], "Burger") should return ["Pizza", "Tacos", "Burger"].
  • You should define a function addLunchToStart.
  • The function addLunchToStart should have two parameters.
  • addLunchToStart(lunches, "Sushi") should log "Sushi added to the start of the lunch menu." to the console.
  • addLunchToStart(["Burger", "Sushi"], "Pizza") should return ["Pizza", "Burger", "Sushi"].
  • You should define a function removeLastLunch.
  • The function removeLastLunch should have one parameter.
  • When the input array is empty, removeLastLunch should log "No lunches to remove." to the console.
  • removeLastLunch(["Stew", "Soup", "Toast"]) should log "Toast removed from the end of the lunch menu." to the console.
  • removeLastLunch(["Sushi", "Pizza", "Noodles"]) should return ["Sushi", "Pizza"].
  • You should define a function removeFirstLunch.
  • The function removeFirstLunch should have a single parameter.
  • When the input array is empty, removeFirstLunch should log "No lunches to remove." to the console.
  • removeFirstLunch(["Salad", "Eggs", "Cheese"]) should log "Salad removed from the start of the lunch menu." to the console.
  • removeFirstLunch(["Sushi", "Pizza", "Burger"]) should return ["Pizza", "Burger"].
  • addLunchToEnd, addLunchToStart, removeLastLunch, and removeFirstLunch should return the same array passed as an argument after updating it.
  • You should define a function getRandomLunch.
  • The function getRandomLunch should have a single parameter.
  • When the input array is empty, getRandomLunch should log "No lunches available." to the console.
  • When the input array is not empty, it should log "Randomly selected lunch: [Lunch Item]" to the console.
  • The getRandomLunch function should not change the array passed to it as an argument.
  • You should define a function showLunchMenu.
  • The function showLunchMenu should have a single parameter.
  • When the input array is empty, it should log "The menu is empty." to the console.
  • showLunchMenu(["Greens", "Corns", "Beans"]) should log "Menu items: Greens, Corns, Beans" to the console.
  • showLunchMenu(["Pizza", "Burger", "Fries", "Salad"]) should log "Menu items: Pizza, Burger, Fries, Salad" to the console.
