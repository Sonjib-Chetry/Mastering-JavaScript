Welcome to my **90 Days of JavaScript Revision Challenge**!

Please read the script file and understand the project and try to build your own.

In this mini project, you will practice how to work with arrays by building a shopping list. You will review how to add and remove elements from an array using methods like push, pop, shift, and unshift.

User Stories

  • You should begin by logging "Grocery shopping list" to the console.
  • You should declare a constant shoppingList initialized as an empty array.
  • You should log a message about wanting fruit, then use push to add "Apples" to the list.
  • You should have a function getShoppingListMsg that returns a formatted string of the current shopping list.
  • You should add more items to the list using push, and print the updated list after each change.
  • You should log a message about needing cooking oil, then use unshift to add "Vegetable Oil" to the beginning of the list.
  • You should add multiple snack items at once using push.
  • You should log a concern about junk food, then remove the last item using pop.
  • You should log about adding dessert and use unshift to add "Chocolate Cake" at the beginning.
  • You should log a health-related message and remove the first item using shift.
  • You should update the first item in the list directly to "Canola Oil" and log the final list.

Test Cases

  1. shoppingList should be an array.
  2. After adding "Apples", shoppingList should contain ["Apples"].
  3. After pushing "Grapes", shoppingList should contain ["Apples", "Grapes"].
  4. After unshifting "Vegetable Oil", shoppingList should contain ["Vegetable Oil", "Apples", "Grapes"].
  5. After pushing 3 snack items, the list should include "Popcorn", "Beef Jerky", and "Potato Chips".
  6. After pop(), "Potato Chips" should be removed from the end of the list.
  7. After unshifting "Chocolate Cake", it should appear at the beginning of the list.
  8. After shifting, "Chocolate Cake" should be removed from the list.
  9. After setting shoppingList[0] = "Canola Oil", the first item should be "Canola Oil".
