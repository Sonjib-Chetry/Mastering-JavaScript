Welcome to my **90 Days of JavaScript Revision Challenge**!

Please read the script file and understand the project and try to build at your own.

One common aspect of web development is learning how to fetch data from an external API, then work with asynchronous JavaScript. In this mini project you will practice how to use the fetch method, dynamically update the DOM to display the fetched data and paginate your data so you can load results in batches.
