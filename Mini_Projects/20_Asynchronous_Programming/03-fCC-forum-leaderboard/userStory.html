  • You should have a function named timeAgo that takes a timestamp in the ISO 8601 format as the argument.
  • The timeAgo function should compute the time difference between the time passed as an argument and the current time and return:
    • xm ago if the amount of minutes that have passed is less than 60.
    • xh ago if the amount of hours that have passed is less than 24.
    • xd ago otherwise.
  • You should have a function named viewCount that takes the number of views of a post as the argument.
  • If the value of views is greater than or equal to 1000, viewCount should return the value divided by 1000, rounded down, with k appended. Otherwise, return the views value.
  • You should have a function named forumCategory that takes the id of a selected category as the argument.
  • forumCategory should verify that the id is a property of the allCategories object and return an anchor element string with:
    • Text: category key
    • Class: category + className of selected category
    • Href: <forumCategoryUrl>/<className>/<id>
    If id does not exist in allCategories, use General with className general.
  • You should have a function named avatars that takes two arrays: posters and users.
  • avatars should return a string of joined <img> elements for each poster:
    • alt = name of user
    • src = avatar_template with {size} replaced by 30
    • If avatar_template is relative, prefix with <avatarUrl>
  • You should have a function named showLatestPosts that takes one parameter (the JSON object).
  • showLatestPosts should extract users and topic_list from the argument and process topics array.
  • #posts-container innerHTML should be set to a string of <tr> elements, one for each topic, each containing 5 <td>:
    1. First <td>: two anchors â†’ post title and category (via forumCategory).
    2. Second <td>: a <div class="avatar-container"> with avatars (via avatars).
    3. Third <td>: replies count (posts_count - 1).
    4. Fourth <td>: views (via viewCount).
    5. Fifth <td>: time since last activity (via timeAgo).
  • You should have an async function named fetchData.
  • fetchData should request data from https://cdn.freecodecamp.org/curriculum/forum-latest/latest.json, call showLatestPosts with the parsed JSON, and log any error using console.log.
