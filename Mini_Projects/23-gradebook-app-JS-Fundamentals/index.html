Welcome to my **90 Days of JavaScript Revision Challenge**!

Please read the script file and understand the project and try to build your own.

For this mini project, you'll create a gradebook app. You'll practice conditionals to determine the student's grade based on their score.

User Stories – Grading System Functions

  • You should define a function named getAverage that:
    • Takes an array of test scores as a parameter.
    • Returns the average score.
  • You should define a function named getGrade that:
    • Takes a student score as a parameter.
    • Returns a letter grade based on the following scale:
    Score Range Grade
    100 A+
    90 - 99 A
    80 - 89 B
    70 - 79 C
    60 - 69 D
    0 - 59 F
  • You should define a function named hasPassingGrade that:
    • Takes a score as a parameter.
    • Returns true if the grade is anything other than "F", otherwise returns false.
    • Uses the getGrade function to determine the grade.
  • You should define a function named studentMsg that:
    • Takes an array of scores and a student score as parameters.
    • Returns a message in this format:
    • "Class average: average-goes-here. Your grade: grade-goes-here. You passed the course."
      OR
      "Class average: average-goes-here. Your grade: grade-goes-here. You failed the course."
    • Uses getAverage and getGrade to compute the average and grade.

Test Cases – Grading System Functions

  • getAverage Function
    • Function named getAverage should exist.
    • It should return a number.
    • getAverage([92, 88, 12, 77, 57, 100, 67, 38, 97, 89]) → 71.7
    • getAverage([45, 87, 98, 100, 86, 94, 67, 88, 94, 95]) → 85.4
    • getAverage([38, 99, 87, 100, 100, 100, 100, 100, 100, 100]) → 92.4
    • getAverage([10, 20, 30, 40, 55, 65, 75, 83]) → 47.25
    • getAverage([10, 20, 30, 40, 50, 60, 70, 97]) → 47.125
  • getGrade Function
    • Function named getGrade should exist.
    • getGrade(100) → "A+"
    • getGrade(90) → "A"
    • getGrade(85) → "B"
    • getGrade(75) → "C"
    • getGrade(65) → "D"
    • getGrade(40) → "F"
  • hasPassingGrade Function
    • Function named hasPassingGrade should exist.
    • It should return a boolean.
    • hasPassingGrade(95) → true
    • hasPassingGrade(45) → false
  • studentMsg Function
    • Function named studentMsg should exist.
    • Should return a full message based on score and grade:
      • studentMsg([92, 88, 12, 77, 57, 100, 67, 38, 97, 89], 37) → "Class average: 71.7. Your grade: F. You failed the course."
      • studentMsg([56, 23, 89, 42, 75, 11, 68, 34, 91, 19], 100) → "Class average: 50.8. Your grade: A+. You passed the course."
      • studentMsg([12, 22, 32, 42, 52, 62, 72, 92], 85) → "Class average: 48.25. Your grade: B. You passed the course."
      • studentMsg([15, 25, 35, 45, 55, 60, 70, 60], 75) → "Class average: 45.625. Your grade: C. You passed the course."
