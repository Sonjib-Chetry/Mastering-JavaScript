For this mini project you will use the audio element to build a drum machine.

  1. You should have a <div> element with an ID of drum-machine that contains all other elements.
  2. Inside #drum-machine, there must be a <div> with an ID of pad-bank.
  3. Inside #drum-machine, include a <p> element with an ID of display.
  4. Inside #pad-bank, create 9 clickable elements with:
    • Class: drum-pad
    • Unique ID describing the sound
    • InnerText: one of these keys in order â€” Q, W, E, A, S, D, Z, X, C
  5. Each .drum-pad should contain an <audio> element with:
    • Class: clip
    • Unique id that matches its parent's text (e.g. Q, W, E...)
    • src pointing to the corresponding sound file
  6. When a .drum-pad is clicked, it should:
    • Play the audio clip
    • Update #display with a unique sound name (e.g. "Heater 1")
  7. When a user presses a key (Q, W, E, A, S, D, Z, X, C), it should:
    • Trigger the associated .drum-pad
    • Play the audio clip
    • Update #display text
