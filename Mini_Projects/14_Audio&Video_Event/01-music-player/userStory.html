In this mini project, you'll code a basic MP3 player using HTML, CSS, and JavaScript. The project covers fundamental concepts such as handling audio playback, managing a playlist, implementing play, pause, next, and previous functionalities and dynamically update your user interface based on the current song.

  1. You must create an instance of Audio and use it to manage playback.
  2. Create a playlist array allSongs with each object having id, title, artist, duration, and src.
  3. Track current song and playback time using a userData object.
  4. When the #play button is clicked:
    • If no song is selected, play the first song.
    • If a song is already playing, resume from the paused time.
  5. When a playlist button is clicked, play that song from the beginning.
  6. The playSong(id, start = true) function should:
    • Set audio.src, update title and artist display, and play the song.
  7. The #pause button should:
    • Pause playback and store the current time.
    • Remove the playing class from the play button.
  8. #previous button should play the previous song. If none, play the first.
  9. #next button should play the next song. If itâ€™s the last, reset the player.
  10. When a song ends, automatically play the next song.
  11. setPlayerDisplay() should update the song name and artist in the player UI.
  12. highlightCurrentSong() should mark the active playlist button using aria-current="true".
  13. setPlayButtonAccessibleText() should set aria-label of the play button dynamically based on the song.
