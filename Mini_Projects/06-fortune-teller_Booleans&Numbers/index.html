Welcome to my **90 Days of JavaScript Revision Challenge**!

Please read the script file and understand the project and try to build your own.

In this mini project, you'll build a fortune teller by randomly selecting a fortune from the available fortunes. You'll practice how to work with the Math.random() method and the Math.floor() method to generate random numbers.

User Stories:

  1. You should declare five variables: fortune1, fortune2, fortune3, fortune4, and fortune5, and assign each a string value.
  2. You should declare a variable randomNumber and assign it a random integer from 1 to 5 (inclusive).
  3. You should declare a variable selectedFortune and assign it one of the five fortune strings based on the value of randomNumber:
    • If randomNumber is 1, assign fortune1.
    • If randomNumber is 2, assign fortune2.
    • If randomNumber is 3, assign fortune3.
    • If randomNumber is 4, assign fortune4.
    • If randomNumber is 5, assign fortune5.
  4. You should log the value of selectedFortune to the console.

Tests

  • You should initialize fortune1 with a string value.
  • You should initialize fortune2 with a string value.
  • You should initialize fortune3 with a string value.
  • You should initialize fortune4 with a string value.
  • You should initialize fortune5 with a string value.
  • You should use the Math.random() method to generate a random number.
  • You should generate a random number between 1 and 5, inclusive, and assign it to the variable randomNumber.
  • You should have a selectedFortune variable that is assigned a value based on the value of randomNumber.
  • The randomNumber should correspond to its fortune. For example, if randomNumber is 1, the selectedFortune should be equal to fortune1 and so on.
  • You should output the selectedFortune to the console.
