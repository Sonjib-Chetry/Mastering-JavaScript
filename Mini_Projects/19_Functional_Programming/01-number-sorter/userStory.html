In computer science, there are fundamental sorting algorithms that all developers should learn. In this number sorter project, you'll learn how to implement and visualize different sorting algorithms like bubble sort, selection sort, and insertion sort ‚Äì all with JavaScript. This project will help you understand the fundamental concepts behind these algorithms, and how you can apply them to sort numerical data in web applications.

üìù User Stories

  • Create a button element with the id sort. This button will trigger the sorting when clicked.
  • Create multiple dropdown elements (HTML <select>) with the class values-dropdown. Each dropdown should allow the user to choose a number.
  • Create output elements with ids in the format output-value-i (example: output-value-0, output-value-1). These elements will display the sorted numbers.
  • In JavaScript, write a function sortInputArray(event) that:
    • Prevents the default button behavior using event.preventDefault().
    • Retrieves all dropdown values with document.getElementsByClassName("values-dropdown").
    • Converts them into an array of numbers.
    • Sorts the array using one sorting algorithm (bubbleSort, selectionSort, or insertionSort).
    • Calls a helper function updateUI(sortedArray) to display the sorted results.
  • Create a helper function updateUI(array) that:
    • Loops through the sorted array.
    • Finds the matching output-value-i element.
    • Updates its innerText with the number at that index.
  • Implement at least three sorting algorithms in separate functions:
    • bubbleSort(array): Compare and swap adjacent elements until sorted.
    • selectionSort(array): Find the minimum element in the unsorted part and swap it into place.
    • insertionSort(array): Insert each element into its correct position in the sorted portion.
  • Attach an event listener to the sort button so that sortInputArray runs when clicked.
