Functional Programming is a popular approach to software development. In Functional Programming, developers organize code into smaller functions, then combine those functions to build complex programs. In this spreadsheet application project, you'll learn about parsing and evaluating mathematical expressions, implementing spreadsheet functions, handling cell references, and creating interactive web interfaces. You'll learn how to dynamically update the page based on user input. This project will cover concepts like the map(), find(), and includes() methods and the parseInt() function.

üìù User Stories

  • You should dynamically generate a spreadsheet grid of cells:
    • Columns labeled from A to J.
    • Rows labeled from 1 to 99.
    • Each cell should be an <input> element with a unique id (e.g., A1, B2).
  • You should implement formulas using an = sign at the beginning of a cell value.
  • You should support basic arithmetic operators:
    • + for addition
    • - for subtraction
    • * for multiplication
    • / for division
  • You should implement spreadsheet-like functions:
    • sum(...) ‚Üí Returns the sum of numbers.
    • average(...) ‚Üí Returns the average.
    • median(...) ‚Üí Returns the median.
    • even(...) ‚Üí Filters even numbers.
    • someeven(...) ‚Üí True if at least one number is even.
    • everyeven(...) ‚Üí True if all numbers are even.
    • firsttwo(...) ‚Üí Returns the first two numbers.
    • lasttwo(...) ‚Üí Returns the last two numbers.
    • has2(...) ‚Üí Returns true if the list contains the number 2.
    • increment(...) ‚Üí Increments each number by 1.
    • random(x,y) ‚Üí Returns a random number between x and y.
    • range(x,y) ‚Üí Returns a numeric range between x and y.
    • nodupes(...) ‚Üí Removes duplicates from the list.
  • You should support cell ranges:
    • For example: =sum(A1:A5) should sum values from A1 through A5.
    • For example: =average(B2:C4) should calculate the average of that 2D range.
  • You should update cell values automatically when a formula is entered (only if input starts with =).
  • You should prevent circular references (a cell referencing itself).
