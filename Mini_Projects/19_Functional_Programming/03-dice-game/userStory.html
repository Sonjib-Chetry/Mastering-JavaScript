Dice Game - Instructive User Stories & Test Cases

User Stories

  • Create 5 dice elements (class .die) to display dice values.
  • Create a score options section (#score-options) containing radio buttons and span labels for score display.
  • Display game stats:
    • Current round (#current-round)
    • Current rolls (#current-round-rolls)
    • Total score (#total-score)
  • Create a history log (#score-history) to list each round’s achieved score.
  • Add buttons:
    • #roll-dice-btn: Roll dice (max 3 times per round)
    • #keep-score-btn: Submit chosen score
    • #rules-btn: Show/hide rules (.rules-container)
  • When "Roll Dice" is clicked:
    • Generate 5 random values (1–6).
    • Update dice display.
    • Enable relevant score options:
      • Three of a Kind → sum of dice
      • Four of a Kind → sum of dice
      • Full House → 25 points
      • Small Straight → 30 points
      • Large Straight → 40 points
      • Fallback → 0 points
  • When "Keep Score" is clicked:
    • Selected score is added to total score.
    • Round increases by 1 (max 6 rounds).
    • Score is added to history.
    • If 6 rounds are completed → show "Game Over" and reset game.
  • Rules button toggles rules display with text changing between Show rules and Hide rules.
  • Game resets to initial state after completion or manual reset.

Test Cases

  • Initial State: Dice show 0, total score = 0, round = 1, rolls = 0.
  • Rolling Dice: Clicking #roll-dice-btn generates 5 numbers between 1–6 and increments roll counter.
  • Roll Limit: After 3 rolls, further clicks on #roll-dice-btn show an alert.
  • Three of a Kind: If at least 3 dice show the same value → radio option for "Three of a Kind" is enabled with sum score.
  • Four of a Kind: If at least 4 dice match → "Four of a Kind" is enabled with sum score.
  • Full House: If 3 dice match one number and 2 match another → "Full House" option is enabled with 25 points.
  • Straights:
    • Small straight (4 consecutive numbers) → 30 points option enabled.
    • Large straight (5 consecutive numbers) → both 30 and 40 points options enabled.
  • Fallback: Always enable "0 points" option in case no other score applies.
  • Keeping Score: Selecting a score and clicking #keep-score-btn adds score to total and round increases by 1.
  • History Log: Each chosen score is appended to #score-history list.
  • Game Over: After round 6, an alert shows final score and game resets to round 1, score 0.
  • Rules Toggle: Clicking #rules-btn alternates rules container visibility and button label.
  • Reset Game: After game over → dice reset to 0, score reset to 0, round reset to 1, rolls reset to 0, history cleared.
