Dessert Shop Cart - JavaScript Development Guide

Overview

This project implements a dessert shopping cart using JavaScript. You will dynamically display products, allow users to add items to their cart, update totals, and clear the cart. HTML and CSS are already provided — you will write only the JavaScript.

Step-by-Step Instructions

1. Select DOM Elements

Create variables using document.getElementById() to store references to HTML elements you will interact with. You must create the following:

  • cartContainer → The entire cart UI section (#cart-container).
  • productsContainer → Where cart items will be listed (#products-container).
  • dessertCards → Container holding all dessert product cards (#dessert-card-container).
  • cartBtn → Button to show/hide the cart (#cart-btn).
  • clearCartBtn → Button to remove all items from cart (#clear-cart-btn).
  • totalNumberOfItems → Span showing total item count (#total-items).
  • cartSubTotal → Span showing subtotal price (#subtotal).
  • cartTaxes → Span showing tax amount (#taxes).
  • cartTotal → Span showing total amount (#total).
  • showHideCartSpan → Span changing text between “Show” and “Hide” (#show-hide-cart).

Also create a variable isCartShowing and set it to false initially to track cart visibility.

2. Create Products Array

Create a constant products — an array of objects. Each object should have:

  • id (number) → Unique identifier for the product.
  • name (string) → Dessert name.
  • price (number) → Price in dollars.
  • category (string) → Dessert category (e.g., Cupcake, Macaron).

3. Render Product Cards

Loop through products and append HTML to dessertCards.innerHTML. Each card should include:

  • Product name in an <h2>.
  • Price prefixed with "$".
  • Category label.
  • “Add to cart” button with:
    • class="add-to-cart-btn"
    • id set to the product’s id

4. Create ShoppingCart Class

The ShoppingCart class will handle cart logic. It should include:

  • Properties:
    • items → array storing all added products.
    • total → numeric value for the grand total.
    • taxRate → set to 8.25 (percentage).
  • Methods:
    • addItem(id, products) → finds product by id, adds it to items, and updates quantity in DOM if already present.
    • getCounts() → returns total number of items in cart.
    • clearCart() → confirms with user, clears items, resets totals, and empties cart display.
    • calculateTaxes(amount) → returns calculated tax for given subtotal.
    • calculateTotal() → updates #subtotal, #taxes, and #total based on items.

5. Add Event Listeners

  • For each “Add to cart” button:
    • On click, call cart.addItem() with the product ID.
    • Update #total-items with cart.getCounts().
    • Call cart.calculateTotal() to refresh totals.
  • For #cart-btn:
    • Toggle isCartShowing.
    • Change #show-hide-cart text accordingly.
    • Show or hide #cart-container.
  • For #clear-cart-btn:
    • Bind cart.clearCart so it runs with correct this context.
