Please read the script file and understand the project and try to build at your own.

In this mini project, you will implement loops to repeat a string a specified number of times.

User Story

You need to create a function named repeatStringNumTimes that repeats a string a given number of times.

Requirements:

  • Create a function named repeatStringNumTimes.
  • The function should take two parameters:
    • string: the text to repeat
    • number: how many times to repeat
  • If the number is <= 0, return an empty string.
  • The function should always return a string.
  • Do not use the built-in repeat() method.
  • Use a for loop or while loop to build the repeated string.

Test Cases

  • You should create a function named repeatStringNumTimes.
  • repeatStringNumTimes should take two parameters.
  • The function should always return a string.
  • repeatStringNumTimes("*", 3) → "***".
  • repeatStringNumTimes("abc", 3) → "abcabcabc".
  • repeatStringNumTimes("abc", 4) → "abcabcabcabc".
  • repeatStringNumTimes("abc", 1) → "abc".
  • repeatStringNumTimes("*", 8) → "********".
  • repeatStringNumTimes("abc", -2) → "" (empty string).
  • repeatStringNumTimes("abc", 0) → "" (empty string).
  • The built-in repeat() method should not be used.
