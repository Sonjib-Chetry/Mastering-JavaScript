Welcome to my **90 Days of JavaScript Revision Challenge**!

Please read the script file and understand the project and try to build at your own.

In this mini project, you'll create a function that looks up profile information.

User Story

You need to create a function named lookUpProfile that looks up information in a given contacts array.

Requirements:

  • A global contacts array will be provided, containing multiple objects. Each object represents a contact with properties like firstName, lastName, number, and likes.
  • Create a function named lookUpProfile that accepts two parameters:
    • name (string): the first name of the contact to search for.
    • prop (string): the property of the contact you want to retrieve.
  • If the contact with the given name exists and has the requested property, return the property’s value.
  • If no contact with the given name exists, return "No such contact".
  • If the contact exists but the property does not exist on it, return "No such property".

Test Cases

  • The contacts array should be present in the global scope.
  • You should have a function named lookUpProfile.
  • lookUpProfile("Kristian", "lastName") → "Vos".
  • lookUpProfile("Sherlock", "likes") → ["Intriguing Cases", "Violin"].
  • lookUpProfile("Harry", "likes") should return an array.
  • lookUpProfile("Bob", "number") → "No such contact".
  • lookUpProfile("Bob", "potato") → "No such contact".
  • lookUpProfile("Akira", "address") → "No such property".
