Welcome to my **90 Days of JavaScript Revision Challenge**!

Please read the script file and understand the project and try to build your own.

In this mini project, you'll build a factorial calculator. You'll practice using loops and conditionals to calculate the factorial of a number.

User Stories – Factorial Calculator

  • You should declare a variable num and assign it a number between 1 and 20 inclusive.
  • Create a function named factorialCalculator that takes a number as an argument and returns the factorial of that number.
  • Inside the function, declare a variable result and assign it the value of 1.
  • Using a loop (for, while, or do...while), multiply the result variable by each number from 1 to the input number (inclusive), updating result in each iteration.
  • You should call the factorialCalculator function with num as the argument and assign the returned value to a variable named factorial.
  • You should store the final output message in the format Factorial of [num] is [factorial] and assign it to the variable resultMsg.
  • You should output the value of resultMsg to the console.

Test Cases – Factorial Calculator

  • You should have a num variable.
  • You should assign a value to the num variable.
  • The value of num should be between 1 and 20.
  • The num value should be a number.
  • You should have a function named factorialCalculator.
  • The factorialCalculator function should take a number as an argument.
  • The factorial of 5 should return 120.
  • The factorial of 7 should return 5040.
  • You should call your factorialCalculator function with the num variable as the argument.
  • You should define a factorial variable and assign the result of the factorialCalculator function to it.
  • Your factorialCalculator should produce the correct result.
  • Your resultMsg should contain a string in the format Factorial of [num] is [factorial].
  • You should output the value of resultMsg to the console.
