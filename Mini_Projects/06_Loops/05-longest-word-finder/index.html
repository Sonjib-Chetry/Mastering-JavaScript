Welcome to my **90 Days of JavaScript Revision Challenge**!

Please read the script file and understand the project and try to build at your own.

In this mini project, you will use JavaScript loops to find the length of the longest word in the given sentence.

User Story

You need to create a function called findLongestWordLength that processes a sentence and finds the length of the longest word inside it.

Requirements:

  • You should create a function named findLongestWordLength.
  • The function should take a single parameter: a string (the sentence to check).
  • The function should break the string into individual words.
  • It should compute the length of each word.
  • It should return the length of the longest word as a number.

Test Cases

  • You should have a function named findLongestWordLength.
  • findLongestWordLength should have a single parameter.
  • findLongestWordLength("The quick brown fox jumped over the lazy dog") → returns 6.
  • findLongestWordLength("May the force be with you") → returns 5.
  • findLongestWordLength("Google do a barrel roll") → returns 6.
  • findLongestWordLength("What is the average airspeed velocity of an unladen swallow") → returns 8.
  • findLongestWordLength("What if we try a super-long word such as otorhinolaryngology") → returns 19.
